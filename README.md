# vimrc
## Installation


```
git clone https://github.com/rhattox/vimrc.git --depth 1 &&\
ln -s ~/vimrc/.vimrc ~/.vimrc &&\
ln -s ~/vimrc/.vim ~/.vim &&\
rm -rf ~/vimrc/.git
```
## Setup

```
# Enter on vim
$ vim

# Install all plugins
:PlugInstall

# Close vim
# Open it again
```


## Clean Up
```
rm -rf ~/.vim
rm -rf ~/.vimrc
```
